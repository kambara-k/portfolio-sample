Javaの基本構文

:contentReference[oaicite:0]{index=0} はオブジェクト指向プログラミング言語で、 Webアプリ、業務システム、Android開発など幅広く使われる。 クラスを中心に構造を作るのが特徴である。 

						<-- 基本的なプログラム構造 -->
						public class Main {
						    public static void main(String[] args) {
						        System.out.println("Hello Java");
						    }
						}
						
						<-- 変数の宣言 -->
						int number = 10;
						String text = "こんにちは";
						
						<-- 条件分岐 -->
						if (number > 5) {
						    System.out.println("5より大きい");
						} else {
						    System.out.println("5以下");
						}
						
						<-- 繰り返し（for文） -->
						for (int i = 0; i < 5; i++) {
						    System.out.println(i);
						}
						
						<-- メソッド定義 -->
						public static int add(int a, int b) {
						    return a + b;
						}

Javaは「クラス」と「メソッド」を基本単位として構成される。 変数・条件分岐・繰り返しを組み合わせて処理を記述する。 

						<-- オブジェクト指向の基本 -->
						class Person {
						    String name;
						
						    void greet() {
						        System.out.println("Hello " + name);
						    }
						}
						
						Person p = new Person();
						p.name = "Taro";
						p.greet();
